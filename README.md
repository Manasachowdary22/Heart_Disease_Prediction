# Heart Disease Prediction

This project uses machine learning to predict the risk of heart disease using clinical features like age, cholesterol, and blood pressure.
Models such as Logistic Regression, Random Forest, and XGBoost are used for classification.
The model helps demonstrate how data analysis and ML can support early heart disease detection.
## Technologies Used

* Python
* Pandas & NumPy
* Scikit-learn
* XGBoost
* Matplotlib & Seaborn
* Google Colab
